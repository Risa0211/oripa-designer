## 主要操作フロー

### ① 競合商品を取り込む（週1〜数回）

**🎴 トレカセンター商品一覧** タブ:
- **🔄 トレカセンター販売中の最新を取得** → カテゴリ選択 → 押す
- 新規追加分の件数と詳細が表示される
- タイトルに「新規限定」「初回限定」を含むものは **自動で🆕新規ガチャ一覧に振り分け**

**🎲 DOPA商品一覧** タブ:
- **🔄 DOPAから最新一覧を取込** を押す
- DOPAのpull_restrictionフラグ判定で **自動で🆕新規ガチャ一覧 / 🎰有料ガチャ一覧へ振り分け**
- 振り分けが違う場合は **🔧 振り分け編集モード** トグル → チェックボックスで修正 → 💾保存

### ② 設計する（📋 景品設計タブ）

**方法A: URL貼り付け（最速）**
- 「🔗 商品URLを貼って即取込」欄に商品URLを貼り付け → **📥 URLから取込**
  - 対応: `https://japan-toreca.com/oripa/...` / `https://dopa-game.jp/pokemon/gacha/...`
- タイトル / 単価 / 総口数 / 賞構成 / カード明細 が**全自動展開**

**方法B: 商品リストから選択**
- 「トレカセンター商品No.」「🎲 DOPA商品から」「🎰 有料ガチャから」「🆕 新規ガチャから」のいずれかで選択
- 📥 読み込み

**方法C: 商品一覧タブから転送**
- 🎴トレカセンター商品一覧 / 🎲DOPA商品一覧 で **行クリック** → 「📋 景品設計に転送」

### ③ 編集と計算

景品明細が展開されたら：

- ヘッダで **単価 / 総口数 / 引く権利の事前課金額（有料ガチャ用）/ 一括上乗せ倍率** を調整
- カード行で各カードの実価値・上乗せ倍率を編集

**🎯 在庫スプシにあるカードはスニダンURL+実価値が自動セット済み**（手入力不要）

- 在庫にない場合のみ、各行の「snkrdunk URL」欄に商品ページURLを貼って **🔄 全行の買取価格を取得**
- 取得結果は **カードマスタDB** に保存され、次回以降は自動入力

リアルタイムに以下が計算表示：
- 売上 / 仕入れ合計 / 粗利 / 顧客還元率 / 実還元率 / 上乗せ差分

## 用語の整理（誤りを防ぐため）

| 用語 | 意味 |
|---|---|
| **通常ガチャ** | pt消費型ガチャ（DOPA/トレカセンター共通の基本形式） |
| **新規ガチャ** | 登録後X日限定 / 24時間限定など、新規ユーザー限定 |
| **有料ガチャ** | **○○円課金者だけ引ける**条件付き限定ガチャ |
| **引く権利の事前課金額** | 有料ガチャを引くために事前に必要な課金額（円） |
| **売上計算** | 単価 × 総口数 + 引く権利の事前課金額（有料時のみ加算） |
| **実価値/枚** | スニダン買取価格ベースの「運営の実コスト」 |
| **顧客還元率** | 表示PT合計 ÷ 売上（顧客が見る還元率） |
| **実還元率** | 仕入れ合計 ÷ 売上（運営の本当の還元率） |

## カード明細の自動入力ルール

景品明細を読み込んだ時、各カードについて以下の順で **実価値とスニダンURL** を自動セット：

1. **在庫スプシ**（PSA10在庫 / BOX在庫）から名前+グレード一致を探す → **スニダンURL + 仕入価格(or 相場)**
2. なければ **カードマスタDB**（リサーチDB内のタブ）を見る
3. なければ 空欄

> 在庫スプシには既に485件分のスニダンURLが登録されているため、**よく出るカードは即自動入力**されます。

### URL空欄を自動検索（DuckDuckGo経由・ベストエフォート） 🆕

景品明細展開後、上部の **🔎 URL空欄を自動検索** ボタンで:
- URL空欄行のカード名+レアで Web検索（DDG→Bingフォールバック）
- `site:snkrdunk.com` で絞り込み、最初の候補URLを自動セット
- 同時に買取価格も取得・カードマスタDBに保存

連続検索すると検索エンジン側のBot検出に当たる場合があります（その時は数分待ってリトライ）。
ヒットしないカードは手動で「snkrdunk URL」列に貼り付け→**🔄 全行の買取価格を取得**。

## トラブル時

### Q. 「gspread.exceptions.APIError」が出る
→ Sheets API のクォータ瞬間超過。リロードすれば数秒〜数分で復活します（自動リトライ実装済み）。頻発する場合は時間を置いてから再試行。

### Q. 在庫にあるカードなのにスニダンURLが入らない
→ 在庫スプシの「カード名」と景品明細のカード名の表記揺れの可能性。名前の一部・空白・全半角の違いを確認してください。

### Q. 振り分け（新規限定/有料）が違う
→ 🎲 DOPA商品一覧の **🔧 振り分け編集モード** で人が修正できます。
